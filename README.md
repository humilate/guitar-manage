# 吉他曲谱管理系统

一个基于 Django 的吉他曲谱管理网站，支持曲谱上传、分类管理、搜索筛选、曲谱共享、分类共享、成员协作和高清图片查看功能。

## 功能特性

- **用户系统**：注册、登录、退出
- **曲谱管理**：上传图片格式的曲谱，自定义命名
- **分类管理**：创建、编辑、删除曲谱分类，支持层级分类
- **搜索功能**：按曲谱名称或分类名称搜索
- **曲谱共享**：一键生成分享链接，无需登录即可查看
- **分类共享**：共享整个分类，其他用户可查看该分类下的所有曲谱
- **成员管理**：分类所有者可添加/移除成员，成员可协作编辑曲谱
- **批量操作**：支持跨分页批量修改分类、批量删除曲谱
- **文件夹上传**：支持批量上传 ZIP 文件夹，自动按文件夹名创建分类
- **高清图片查看**：点击曲谱图片可放大查看，支持鼠标滚轮缩放（25% - 500%）、拖动平移、键盘快捷键操作
- **图片下载**：在放大查看模式下可直接下载高清曲谱图片
- **拼音目录**：分类详情页面集成按曲谱名称拼音首字母（A-Z）分类的目录，支持字母导航快速定位
- **响应式设计**：适配桌面和移动设备
- **后台管理**：Django Admin 管理用户和内容
- **页面动画**：平滑的页面切换和卡片加载动画

## 图片查看器功能

### 基本操作
- **点击曲谱图片**：进入全屏放大查看模式
- **关闭查看器**：点击关闭按钮或按 `ESC` 键
- **切换页面**：点击左右箭头或使用键盘方向键 `←` `→`

### 缩放功能
- **鼠标滚轮**：向上滚动放大，向下滚动缩小
- **缩放按钮**：点击右下角 `+` `-` 按钮进行缩放
- **快捷键**：`+`/`=` 键放大，`-` 键缩小，`0` 键重置为 100%
- **缩放范围**：25% - 500%

### 拖动平移
- 当图片放大后，鼠标指针变为移动状态
- 按住鼠标左键拖动可查看图片不同区域

### 下载功能
- 在查看器中点击右下角下载按钮可保存当前页图片
- 支持右键另存为功能

## 拼音目录功能

### 使用方式
1. 进入任意分类详情页面
2. 点击顶部的"📚 目录"展开目录面板
3. 通过字母导航（A-Z/#）快速定位到目标曲谱
4. 点击曲谱名称直接跳转到曲谱详情页

### 技术实现
- 使用 `pypinyin` 库获取曲谱名称的拼音首字母
- 曲谱按拼音首字母分组显示（A-Z，非中文开头的归入 #）
- 支持展开/收起目录面板

## 技术栈

- **后端**：Python + Django 5.0.6
- **数据库**：SQLite（默认）
- **前端**：HTML + CSS + JavaScript（原生）
- **图片处理**：Pillow
- **拼音处理**：pypinyin

## 安装运行

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 数据库迁移

```bash
python manage.py makemigrations
python manage.py migrate
```

### 3. 创建超级用户（可选）

```bash
python manage.py createsuperuser
```

### 4. 启动服务器

```bash
python manage.py runserver
```

### 5. 访问网站

打开浏览器访问：**http://127.0.0.1:8000/sheets/login/**

## 使用指南

### 注册账号
- 访问 `/sheets/register/`
- 填写用户名、邮箱和密码

### 上传曲谱
1. 点击"上传曲谱"按钮
2. 填写曲谱名称
3. 选择分类（可选）
4. 上传曲谱图片
5. 点击"保存"

### 批量上传文件夹
1. 点击"批量上传文件夹"按钮
2. 上传 ZIP 文件
3. 系统自动按文件夹名创建分类

### 创建分类
1. 点击"新建分类"按钮
2. 填写分类名称和描述
3. 点击"保存"

### 共享分类
1. 在分类页面点击"共享分类"按钮
2. 共享后生成分享链接
3. 复制链接发送给朋友

### 管理成员
1. 进入分类页面（仅所有者）
2. 点击"管理成员"按钮
3. 输入用户名添加成员
4. 成员可访问共享分类并编辑自己上传的曲谱

### 查看和放大曲谱
1. 点击曲谱卡片进入详情页
2. 点击曲谱图片进入全屏查看模式
3. 使用鼠标滚轮或按钮进行缩放
4. 拖动查看图片不同区域
5. 点击下载按钮保存高清图片

### 批量操作
1. 在分类页面勾选要操作的曲谱
2. 选择"全选本页"或"选择全部 X 首"
3. 选择目标分类或直接删除

### 搜索曲谱
- 在搜索框输入曲谱名称或分类名称
- 点击"搜索"按钮

### 使用拼音目录
1. 进入分类详情页面
2. 点击顶部"📚 目录"展开
3. 通过字母导航定位曲谱
4. 点击曲谱名称查看详情

| `/sheets/category/<id>/delete/` | 删除分类 | 是 |
| `/sheets/category/<id>/share/` | 切换分类共享状态 | 是 |

| `/sheets/category/<id>/batch/update/` | 批量修改分类 | 是 |
| `/sheets/category/<id>/batch/delete/` | 批量删除 | 是 |
| `/sheets/category/shared/<token>/` | 查看共享分类 | 是（成员/所有者） |
| `/sheets/upload_folder/` | 批量上传文件夹 | 是 |

## 数据模型

### Category（曲谱分类）
| 字段 | 类型 | 说明 |
|------|------|------|
| name | CharField | 分类名称 |
| description | TextField | 分类描述 |
| owner | ForeignKey | 所有者（用户） |
| members | ManyToManyField | 成员（可访问编辑） |
| share_token | UUIDField | 分享令牌（唯一） |
| is_shared | BooleanField | 是否共享 |
| created_at | DateTimeField | 创建时间 |

### GuitarSheet（吉他曲谱）
| 字段 | 类型 | 说明 |
|------|------|------|
| title | CharField | 曲谱名称 |
| category | ForeignKey | 所属分类 |
| owner | ForeignKey | 所有者（用户） |
| share_token | UUIDField | 分享令牌（唯一） |
| is_shared | BooleanField | 是否共享 |
| created_at | DateTimeField | 创建时间 |

### SheetImage（曲谱图片）
| 字段 | 类型 | 说明 |
|------|------|------|
| sheet | ForeignKey | 所属曲谱 |
| image | ImageField | 曲谱图片 |
| page_number | PositiveIntegerField | 页码 |
| created_at | DateTimeField | 上传时间 |

## 部署

### PythonAnywhere 部署

```bash
cd ~/guitar-manage
git pull
python manage.py migrate
```

然后在 Web 页面点击 **Reload** 重新加载应用。

### 注意事项
- PythonAnywhere 虚拟环境需要安装 `pypinyin`：`pip install pypinyin`
- 重载应用：`touch guitar_sheet_project/wsgi.py`
